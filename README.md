# Iron Pulse

Iron Pulse is a collection of web applications and prototypes focusing on interactive user experiences and modern web design to make a "Centered Heart Particle System" and "Face Recognition System".

## Project Structure

The repository contains the following main components:

- **ironpulse-web**: A React application built with Vite. This appears to be the main web interface featuring the Iron Pulse landing page.
- **ironpulse-app**: A TypeScript-based application (Vite).
- **ironpulse-landing**: Contains resources and code related to the project's landing page.

## Getting Started

To run any of the sub-projects, navigate to the directory and install dependencies:

### ironpulse-web

```bash
cd ironpulse-web
npm install
npm run dev
```

### ironpulse-app

```bash
cd ironpulse-app
npm install
npm run dev
```

## Features

- **Modern UI/UX**: Designed with aesthetics in mind, utilizing glassmorphism and smooth animations.
- **Interactive Elements**: Includes particle systems and gesture controls (based on recent development history).
